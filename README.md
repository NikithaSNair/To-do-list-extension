# To-do-list-extension

# Description

A simple Chrome extension to manage your tasks. 
Add tasks to a list.
Delete tasks when completed.
Tasks are saved using chrome.storage.sync so they persist even after closing the browser.
This is a beginner-friendly project that demonstrates Chrome Extension development and basic storage handling.

# Features

Add a task using the + button.
Delete tasks by clicking the x next to each task.
Tasks are automatically saved and loaded from Chrome storage.

# Installation / Usage

Clone or download this repository.
Open Chrome and go to chrome://extensions/.
Enable Developer Mode (top-right corner).
Click Load unpacked and select the project folder.
Click the extension icon in the toolbar to open the To-Do popup.
Add tasks, delete tasks, and see them persist across browser sessions.

# Technologies Used

HTML, CSS, JavaScript
Chrome Extensions API
chrome.storage.sync for storing tasks
